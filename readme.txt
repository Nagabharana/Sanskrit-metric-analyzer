How To go to the Sanskrit metric analyzer home page :

1. Download the flask package and flask_wtf package.
2. Download python files related to the flask frontend.
3. Create a folder in the same path location where you have saved flask files named as 'templates' and store all the HTML files attached with this in that folder.
4. Run the flask files.
5. Copy the localhost name that obtained from running flask file and paste it in google/mozilla firefox search bar.
6. It opens an html home page.
7. Start executing WebApp.

How to download Flask-wtf and flask

In ubuntu and debien

#pip install Flask-WTF
#pip install flask

In windows

$easy_install flask-wtf
$easy_install flask

To run App

$ python app.py
